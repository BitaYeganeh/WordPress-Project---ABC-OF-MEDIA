# THE ABC BOOK OF MEDIA LITERACY  
### A New Primer for Every 9th Grader in Finland

## 📌 Project Overview

The ABC Book of Media Literacy is a WordPress-based campaign website developed using PHP and MySQL.

The website promotes a nationwide Finnish educational initiative providing every ninth grader (ages 14–15) with a physical media literacy primer designed to combat disinformation, misinformation, and hybrid threats in the digital age.

The project highlights the importance of critical thinking, democratic resilience, and responsible media consumption in an increasingly fragmented and AI-driven media landscape.

---

## 🎯 Purpose of the Website

This website was developed to:

- Present the initiative and its mission
- Provide press materials (release, images, videos)
- Offer downloadable resources
- Share background information and FAQs
- Promote awareness of media literacy education

---

## 🌍 About the Initiative

“The ABC Book of Media Literacy” uses the alphabet as a framework to introduce key media literacy concepts — from algorithms to zines — offering an illustrated A–Z guide for navigating today’s complex media environment.

The initiative is led by News Media Finland (NMF) and aims to strengthen democratic resilience by equipping young people with essential critical thinking tools.
